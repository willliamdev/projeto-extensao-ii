# Projeto de Extensão II: Site Institucional

Este projeto faz parte do **Programa de Ação e Difusão Cultural** do curso de Sistemas de Informação (Projeto de Extensão II). O objetivo é aplicar conhecimentos técnicos de desenvolvimento web para fortalecer e digitalizar grupos culturais locais.

## 🎯 Finalidade e Motivação
A ação visa fornecer uma presença digital para uma escola/grupo de dança, facilitando a divulgação de suas atividades e a conexão com a comunidade. Através da tecnologia, buscamos democratizar o acesso à cultura, otimizar a gestão de informações e profissionalizar a imagem do grupo para ampliar suas oportunidades de visibilidade e captação de patrocínios.

## 🌍 Objetivos de Desenvolvimento Sustentável (ODS)
O projeto está alinhado com as metas da Agenda 2030 da ONU:
* **ODS 4 - Educação de Qualidade:** Promover o acesso a informações culturais e facilitar a aprendizagem da arte da dança.
* **ODS 8 - Trabalho Decente e Crescimento Econômico:** Apoiar a profissionalização de grupos culturais, auxiliando na geração de renda e visibilidade local.
* **ODS 11 - Cidades e Comunidades Sustentáveis (Meta 11.4):** Fortalecer os esforços para salvaguardar o patrimônio cultural local.

## Roadmap: Lista de Tarefas 

### Fase 1: Planejamento e Requisitos
- [ ] Entrevista com o grupo de dança para definir seções (Início, Aulas, Eventos, Contato).
- [ ] Coleta de materiais (Logotipo, fotos, textos descritivos).
- [ ] Definição da paleta de cores e identidade visual.

### Fase 2: Estrutura e Estilo (HTML/CSS)
- [ ] Criação da estrutura semântica das páginas.
- [ ] Implementação do layout responsivo (Mobile First).
- [ ] Estilização de componentes (Navegação, botões, rodapé).

### Fase 3: Dinamismo (JavaScript)
- [ ] Implementação de menu *hamburger* para dispositivos móveis.
- [ ] Criação de uma galeria de fotos dinâmica ou carrossel.
- [ ] Formulário de contato com validação de campos.

### Fase 4: Implantação e Finalização
- [ ] Configuração do repositório no GitHub.
- [ ] Ativação do **GitHub Pages** nas configurações do projeto.
- [ ] Teste de performance e acessibilidade.
- [ ] Coleta do depoimento da instituição para o relatório final.

## 🎓 Competências a serem Aplicadas
Este projeto servirá como laboratório prático para as seguintes competências previstas no curso de Sistemas de Informação:  
- Especificação e Projeto: Traduzir as necessidades do grupo de dança em requisitos de software.  
- Desenvolvimento de Soluções de TI: Implementar o site utilizando tecnologias web.
- Comunicação Interpessoal: Atuar diretamente com os responsáveis pelo grupo cultural para coleta de dados e feedback.
